A sample of how to filter a list with existing json.
You are asked to improve the page by the following tasks.
  • Create Product Page.

    1. Show list of products in a 2-column grid like illustration below:



    2. Products list data source is from products.json.

    3. The design is up to you but this page needs to have at least these components with specific ID as it stated on the table below.

      View Component ID
      Product RecyclerView rv_product_list
      Product Image iv_product
      Product Name tv_product_name
      Product Price tv_product_price
      Product Slashed Price tv_product_price_slashed
      Product Discount tv_product_discount
      City tv_shop_city


  • Create Filter BottomSheet.

    1. Create a filter button in the Product Page. When this button is clicked, Filter BottomSheet will be shown so user can configure the filter based on: location and price.

    2. Location filter should use Choice chips, single selection, and only show 3 options:
      (1) The location that appears most frequent in the list
      (2) The second most frequent item in the list, and
      (3) The third most frequent item in the list

      For example:
      Location:
      [Jakarta] [Bali] [Bandung]

      This options should not be hardcoded.

    3. Price Filter should be enable user to input min and maximum price (can use Slider). Min and max price is based on the product list and should not be hardcoded.

      For example:
      Min = 15000 (15000 is the minimum price in the list)

    4. In the bottom sheet, there will be a button to submit the filter configuration. If this submit button is clicked, the product list will reflect the change according to the given input.

    5. This bottomsheet needs to have at least these components with specific ID as it stated on the table below.

      View Component ID
      Chip Group Location cg_location
      Chip chip_location_[idx]
      (as a tag)
      Price Range Slider range_slider_price
      Filter Button btn_filter

      Note for Chip:

      • just for Chip view, please put chip_location_[idx] as a tag.
      • [idx] starts from 0,1,2..
      • Example : chip_location_0, chip_location_1, ...



  • Don't add new library.
